E-Resources

EBSCO Web Discovery Service

Library has subscribed EBSCO Web Discovery Service, which enables the students to access journals and e-resources online.

The unique full text search limiter will return results from e-journals and databases subscribed by EBSCO and Somaiya institutional libraries.

Online Public Access Catalogue (OPAC)

Library is equipped with computerized online public access catalogue (OPAC) for searching availability of books in Somaiya Trust educational institutions

In order to avail the maximum utilization of the library resources, online public access catalogue (OPAC) facility is available on following website.
