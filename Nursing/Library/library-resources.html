All the active book collection is updated in the BOOKWARM Library Management Software database and the Web OPAC is available for the users. The issue and return of books has been activated with the BOOKWARM Library Management Software.

Various types of reports can be generated with the use of above mentioned software which is useful for various committees and inspections from time to time. Book database is created along with user’s database. Issue return process is fully automated and manual cards are kept as supporting documents for students and faculty. Bar-coding used: Yes Book issue and return is carried out with the help of barcode system. Each book and user’s borrower card is bar coded.

Web OPAC (Online Public Access Catalogue) facility is made available through BOOKWARM Library Management Software to know the bibliographical details about the collection. User can use computers or personal mobiles to use OPAC facility. And as it is a Web OPAC, user from any location can search the library collection. Search by giving Title, Author, and domain can be carried out. The college area is also Wi-Fi enabled.

Electronic Resource Management package for e-journals

The library subscribes to MAT e-journals nursing package which includes 12 print + e-journals. As well as In addition to that, link to scholarly open access journals/database is also available on the Library webpage.

A well-equipped Digital Library with 13 nodes having Internet connectivity is housed in the Library for access to E-Resources. As the access facility to e-journals is multi-user and IP address-based, students can access the E-Resources from anywhere in the campus.
